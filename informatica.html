Informática

Página web del proyecto

En la materia de Informática hemos trabajado en la creación de una página web que reúne todas las asignaturas de este proyecto interdisciplinario. En ella incluimos la información, actividades y productos que elaboramos en cada materia, como Ciencias, Español, Inglés, Educación Ambiental, etc.

El objetivo de esta página es integrar todo lo aprendido en un solo espacio digital, demostrando nuestras habilidades en organización, diseño, redacción y trabajo en equipo.

Durante este proceso aprendimos a usar herramientas de diseño web, etiquetas HTML y estilos CSS, además de mejorar la presentación visual de nuestros contenidos.

También añadimos imágenes, enlaces y secciones interactivas para que sea más fácil navegar entre los temas y visualizar lo trabajado en cada área.

Este proyecto nos permitió comprender cómo la tecnología puede ser una herramienta útil para compartir conocimiento de manera creativa y accesible.
